# nemo-set-as-desktop-background
Right-click any image in Nemo to set it as your desktop wallpaper. Only works on GNOME systems.

<img width="1009" height="694" alt="immagine" src="https://github.com/user-attachments/assets/f8782478-cca0-4c1d-9cd6-cc0d23cee352" />

## Setup

Download the file `set-as-desktop-background.nemo_action` from the repository and copy it in `/home/$USER/.local/share/nemo/actions`. Restart Nemo in order to apply changes.
